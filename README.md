# Mencius-frontend

## About

Front-end codebase for [Mencius](https://nopro.studio/mencius) project.

## Building

### Setting up env

create `.env` file likes:

```sh
API_HOST="http://localhost:4200"
CLIENT_ID=""
REDIRECT_URI="http://localhost:8080/#/github_oauth_cb"
ENV="debug"
```

### Starting the server

```sh
yarn install
yarn start
```
